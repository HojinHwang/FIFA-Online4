# FIFA-Online4

## 피파온라인4 데이터 분석 프로젝트

### 기간 : 2021/01 ~ 2021/04

- 넥슨 개발자센터 https://developers.nexon.com/ 에서 피파온라인4 API를 발급 받아 진행하는 데이터 분석 프로젝트
- 나의 경기 내용을 분석하고, 다양한 인사이트를 도출하려 함 (ex. 시간대에 따른 승률 분석)

### 기간: 202408 ~

- 이전에 진행했던 프로젝트를 디벨롭하여 새로운 프로젝트 진행
- Airflow를 활용한 ETL 작업과 Tableau를 활용한 시각화 진행
